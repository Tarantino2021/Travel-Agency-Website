# Travel-Agency-Website

It is a static Travel Agency site with 4 pages using Framer Motion and GSAP for subtle animation and transitions. It is fully responsive. I utilize modern UI/UX designs. 

I utilized this:

 	* ReactJS
  
 	* SASS
  
 	* Javascript
  
 	* Material-Ui
  
 	* Framer Motion
  
 	* React-Hook-Form
  
 	* React-Youtube
  
 	* React-Router-Dom
  
 	* React-Date-Range
